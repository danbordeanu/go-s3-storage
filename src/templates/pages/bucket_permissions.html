Permission Guide

  • Owner: The user who created the bucket. Owners always have full read and write access.
  • Read: Users can view and download objects from this bucket.
  • Write: Users can upload, modify, and delete objects in this bucket.
  • Admins: Admin users always have full access to all buckets regardless of these settings.
